# Cybersecurity Awareness Chatbot (Part 1)

## Overview
This is a C# console-based chatbot designed to educate users on basic cybersecurity practices such as password safety, phishing awareness, and safe browsing.

## Features
- Voice greeting using WAV file
- ASCII styled header
- Personalized user interaction
- Basic cybersecurity responses
- Input validation
- Colored console UI

## How to Run
1. Open the project in Visual Studio
2. Ensure the greeting.wav file is inside the Assets folder
3. Build and run the project

## GitHub Requirements
- Minimum 6 commits
- CI setup using GitHub Actions
- Successful build (green check)

## Topics Covered
- Password Safety
- Phishing Awareness
- Safe Browsing
